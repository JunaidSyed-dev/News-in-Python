# Python News CLI App

A command-line Python application that fetches and displays news headlines using a news API.

## Features

- Fetches latest news headlines
- Displays news in the terminal
- Works with API response data
- Helps practice Python API handling and JSON parsing

## Tech Stack

- Python
- Requests
- JSON
- News API

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/JunaidSyed-dev/News_in_python.git
   ```

2. Open the project folder:

   ```bash
   cd News-in-Python
   ```

3. Install required dependencies:

   ```bash
   pip install requests
   ```

4. Run the Python file:

   ```bash
   python main.py
   ```
